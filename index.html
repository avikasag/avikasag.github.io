Statistical Analysis of Automotive Fuel Economy

For the "Statistical Analysis of Automotive Fuel Economy" project, I utilized Python on Kaggle to explore and analyze large datasets, converting numeric variables into categorical formats, addressing missing data, and standardizing formats for analysis. The exploratory data analysis (EDA) involved data filtration, cleaning, removing duplicates, and unnecessary columns. Hypothesis tests like the Chi-square, Mann-Whitney, and Fisher's Exact Test were conducted. Machine learning models such as Random Forest, KNN Regression, and XGBoost Regression were employed to predict fuel economy metrics, demonstrating proficiency in data science and machine learning techniques.

Attrition AI: Employee Churn Prediction

In the "Attrition AI: Employee Churn Prediction" project, I worked with the IBM dataset from Kaggle to build a predictive model aimed at identifying employees most likely to leave the company. Using machine learning algorithms like Logistic Regression, Random Forest, and XGBoost, supplemented by Cosine similarity, I developed a model with an accuracy rate of 87%. The project was implemented using Python and Jupyter Notebook, designed to assist HR departments in taking proactive measures by accurately predicting potential employee attrition and improving retention strategies.
